# JMAS Operatividad - Actualizaciones

Archivos publicos usados por la app Seguridad Operativa para buscar y descargar actualizaciones.

- `update.json`: version disponible y URL de descarga.
- `Seguridad-Operativa-v1.18.apk`: instalador Android actual.
- `Seguridad-Operativa-v1.17.apk`: version anterior.
- `Seguridad-Operativa-v1.16.apk`: version anterior con inicio automatico y respuesta visual en botones.
- `Seguridad-Operativa-v1.15.apk`: version anterior conservada para respaldo.
